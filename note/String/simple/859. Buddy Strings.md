### 题目
给定两个由小写字母构成的字符串 `A` 和 `B` ，只要我们可以通过交换 `A` 中的两个字母得到与 `B` 相等的结果，就返回 `true` ；否则返回
`false` 。



**示例 1：**

    
    
    **输入：** A = "ab", B = "ba"
    **输出：** true
    

**示例 2：**

    
    
    **输入：** A = "ab", B = "ab"
    **输出：** false
    

**示例 3:**

    
    
    **输入：** A = "aa", B = "aa"
    **输出：** true
    

**示例 4：**

    
    
    **输入：** A = "aaaaaaabc", B = "aaaaaaacb"
    **输出：** true
    

**示例 5：**

    
    
    **输入：** A = "", B = "aa"
    **输出：** false
    



**提示：**

  1. `0 <= A.length <= 20000`
  2. `0 <= B.length <= 20000`
  3. `A` 和 `B` 仅由小写字母构成。
