### 题目
 给定字符串`J` 代表石头中宝石的类型，和字符串 `S`代表你拥有的石头。 `S`
中每个字符代表了一种你拥有的石头的类型，你想知道你拥有的石头中有多少是宝石。

`J` 中的字母不重复，`J` 和 `S`中的所有字符都是字母。字母区分大小写，因此`"a"`和`"A"`是不同类型的石头。

**示例 1:**


​    
    输入: J = "aA", S = "aAAbbbb"
    输出: 3


**示例 2:**


​    
    输入: J = "z", S = "ZZ"
    输出: 0


**注意:**

  * `S` 和 `J` 最多含有50个字母。
  *  `J` 中的字符不重复。

### 参考代码

```go
func numJewelsInStones(J string, S string) int {
	list:= ['z'-'A'+1]int{}
	count:=0

	//遍历J将其中的字母装入哈希表(数组)
	for _,value := range J {
		list[value-'A']++
	}

	//遍历S统计有多少字母在哈希表里
	for _,value := range S {
		if list[value-'A'] != 0 {
			count++
		}
	}

	return count
}
```
